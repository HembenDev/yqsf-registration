📋 About
This portal allows YQSF members in Plateau State to register their details digitally. All submissions are saved automatically to a secure Google Sheet accessible only to YQSF executives.
What the form captures:
Personal information (name, gender, contact details, LGA)
Academic background (institution, graduation year, NYSC status)
Professional details (employment, area of practice, NIQS membership)
Skills and volunteer interest
---
⚙️ How It Works
```
Member fills form → Submits → Data saved to Google Sheet → Admin receives email notification
```
